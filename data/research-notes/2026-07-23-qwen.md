## 1. Qwen3.8 Max — launch, claims, specs

- Preview version ("Qwen3.8-Max-Preview") announced **19 July 2026** (Sunday) at the World Artificial Intelligence Conference (WAIC) in Shanghai; most news coverage published 19–21 July 2026.
  — SOURCE: https://www.marktechpost.com/2026/07/19/alibaba-previews-qwen3-8-max-a-2-4-trillion-parameter-multimodal-model-days-after-moonshots-kimi-k3-open-weight-launch/ ; also https://www.scmp.com/tech/article/3361119/alibaba-says-newest-qwen-ai-model-second-only-anthropics-claude-fable-5

- Alibaba's central claim, quoted directly: **"one of the most powerful models available today, comparable to leading frontier AI models, second only to Fable 5."**
  — SOURCE: https://www.scmp.com/tech/article/3361119/alibaba-says-newest-qwen-ai-model-second-only-anthropics-claude-fable-5

- WSJ phrasing (via Yahoo syndication, Tracy Qu byline): Alibaba describes the model as **"second only to Fable 5, Anthropic's flagship Mythos-class model."**
  — SOURCE: https://finance.yahoo.com/technology/ai/articles/alibaba-says-ai-model-just-053700283.html

- Model size: **2.4 trillion total parameters**, Mixture-of-Experts (MoE) architecture; this is Qwen's first multimodal model above 1 trillion parameters (text, images, video, documents in; text out). **Active parameters per token were NOT disclosed** by Alibaba.
  — SOURCE: https://www.techtimes.com/articles/321158/20260721/alibabas-qwen38-max-claims-second-place-behind-fable-5-no-benchmarks-published.htm ; corroborated https://www.marktechpost.com/2026/07/19/alibaba-previews-qwen3-8-max-a-2-4-trillion-parameter-multimodal-model-days-after-moonshots-kimi-k3-open-weight-launch/

- Context window and output, per a Chinese-language model-card aggregator: input context **~1M tokens** (983,616 in one cited config), max output **~256K tokens** (131,072 in one cited config); "thinking/fast" dual inference modes supported.
  — SOURCE: https://www.datalearner.com/ai-models/pretrained-models/qwen3-8-max-preview ; context figures corroborated at https://www.techtimes.com/articles/321158/20260721/alibabas-qwen38-max-claims-second-place-behind-fable-5-no-benchmarks-published.htm

- Launched **two days after** Moonshot AI's Kimi K3 (2.8T parameters, open-weight), and coverage frames the timing as a direct competitive response.
  — SOURCE: https://www.marktechpost.com/2026/07/19/alibaba-previews-qwen3-8-max-a-2-4-trillion-parameter-multimodal-model-days-after-moonshots-kimi-k3-open-weight-launch/

### Open weights / license status

- **Currently NOT open — API/subscription preview only.** A Chinese-language model card states explicitly: "代码开源状态 · 不开源" (code open-source status: not open source) and "预训练权重开源 · 不开源" (pretrained weights open-source status: not open source).
  — SOURCE: https://www.datalearner.com/ai-models/pretrained-models/qwen3-8-max-preview

- Alibaba has said an open-weight release is **"planned soon"** but has **not** announced a release date, license terms, benchmark results, or additional technical documentation. Every prior Max-tier Qwen model remained closed-source, so an eventual open release (if it happens) would break that pattern.
  — SOURCE: https://www.fonearena.com/blog/487749/alibaba-qwen3-8-max-preview-features.html ; pattern point also at https://www.marktechpost.com/2026/07/19/alibaba-previews-qwen3-8-max-a-2-4-trillion-parameter-multimodal-model-days-after-moonshots-kimi-k3-open-weight-launch/

- XenoSpectrum (analysis outlet) explicitly cautions: **"It would be premature to call Qwen3.8 an open-weight model at this point. What Qwen has announced is a plan to release the weights—there is no distribution file or license yet."**
  — SOURCE: https://xenospectrum.com/en/qwen3-8-open-weight-preview/

### Pricing

- No standard/production API per-token pricing has been published. Chinese model-card page states plainly: "暂无公开的 API 定价信息" (no publicly disclosed API pricing information).
  — SOURCE: https://www.datalearner.com/ai-models/pretrained-models/qwen3-8-max-preview

- Preview access is priced at **"10% of the standard price"** during the trial period, sold via Alibaba's Token Plan subscription.
  — SOURCE: https://www.fonearena.com/blog/487749/alibaba-qwen3-8-max-preview-features.html ; corroborated https://www.marktechpost.com/2026/07/19/alibaba-previews-qwen3-8-max-a-2-4-trillion-parameter-multimodal-model-days-after-moonshots-kimi-k3-open-weight-launch/

- Subscription tiers cited (figures vary slightly by source/date, likely different snapshots of a limited-time promo): "Lite ($6/month), Pro ($68/month at 10% standard rate)" — SOURCE: https://www.techtimes.com/articles/321158/20260721/alibabas-qwen38-max-claims-second-place-behind-fable-5-no-benchmarks-published.htm. Elsewhere: "Personal Edition pricing ranges from $8–$80/month (limited-time discounts to $6–$68/month)" — SOURCE: https://www.fonearena.com/blog/487749/alibaba-qwen3-8-max-preview-features.html. A Chinese Alibaba Cloud developer-community post cites **"个人版低至39元/月"** (personal edition from 39 CNY/month) — SOURCE: https://developer.aliyun.com/article/1749296

- No confirmed USD-per-million-token or CNY-per-million-token API rate for Qwen3.8 Max was found (UNVERIFIED). For comparison, the prior-generation Qwen3.7 Max carries a published Artificial Analysis rate of **$2.50/1M input tokens, $7.50/1M output tokens** (cached input $0.25, -90%) — SOURCE: https://artificialanalysis.ai/models/qwen3-7-max — but this is the older model, not Qwen3.8.

### Availability

- Preview accessible via Alibaba's **Token Plan** subscription, **Qoder** (coding environment) and **QoderWork** (agentic/desktop suite); Token Plan is said to support integration with Qwen Code, OpenClaw, OpenCode, Claude Code, Codex, Cline, Cursor, and Kilo CLI plus their IDE plugins.
  — SOURCE: https://www.fonearena.com/blog/487749/alibaba-qwen3-8-max-preview-features.html

- One outlet frames preview access as principally for **developers in mainland China** via Token Plan/Qoder/QoderWork, with **"no global availability details provided"** at launch — flag as UNVERIFIED whether/how non-China users can access it.
  — SOURCE: https://dataconomy.com/2026/07/20/qwen3-8-24t-parameters-alibaba-ai-model-launch/

## 2. Benchmark claims and the skepticism around them

- **No benchmark table, model card, or independent evaluation has been published for Qwen3.8 Max.** This is the dominant theme across nearly every outlet reviewed.
  — SOURCE: https://www.techtimes.com/articles/321158/20260721/alibabas-qwen38-max-claims-second-place-behind-fable-5-no-benchmarks-published.htm ; also https://xenospectrum.com/en/qwen3-8-open-weight-preview/ ; also https://technode.com/2026/07/21/alibaba-outlines-qwen3-8-with-2-4-trillion-parameters/ ("Alibaba has not yet provided detailed information on the model's architecture, training data, benchmark results or release schedule.")

- Confirmed directly: as of this research, **Artificial Analysis has no listing page for "Qwen3.8 Max"** — its Qwen coverage stops at Qwen3.7 Max and Qwen3 Max Thinking (Preview).
  — SOURCE: https://artificialanalysis.ai/models/qwen3-7-max (checked; no qwen3-8-max equivalent found in search)

- Techtimes states the ranking claim is **absent from Artificial Analysis, Arena.AI (LMArena), and the Hugging Face Open LLM Leaderboard**, and that the "second only to Fable 5" line traces to **"a single X post from Qwen team"** — i.e., Alibaba's own characterization, not a third-party result.
  — SOURCE: https://www.techtimes.com/articles/321158/20260721/alibabas-qwen38-max-claims-second-place-behind-fable-5-no-benchmarks-published.htm

- XenoSpectrum, quoted directly: **"Qwen has not disclosed which tasks were compared, which version of Fable 5 was used, the amount of thinking involved, the agent execution environment, or the number of trials."** Conclusion: the ranking "remains Qwen's own self-assessment."
  — SOURCE: https://xenospectrum.com/en/qwen3-8-open-weight-preview/

- Comparison point raised by XenoSpectrum: Moonshot's Kimi K3 (launched days earlier) disclosed architecture details, API pricing ($3/$15 per million tokens), and evaluation conditions **before** Qwen3.8 released any specifics at all.
  — SOURCE: https://xenospectrum.com/en/qwen3-8-open-weight-preview/

- Historical contrast cited by SiliconANGLE: Qwen3.8's immediate predecessor, **Qwen3.7-Max (May 2026), shipped with a published score of 56.6 on the Artificial Analysis Intelligence Index** and sits "well down LMArena's list, where Fable 5 is No. 1." Qwen3.8 arrived with "no model card, no activated-parameter count and no benchmark data at all." Article's closing line: **"Benchmarks and the promised open weights would settle it, but Alibaba has released neither."**
  — SOURCE: https://siliconangle.com/2026/07/19/alibaba-previews-qwen3-8-claims-second-claude-fable-5/ (Artificial Analysis' own page independently lists Qwen3.7 Max at **57 / #7 of 148** — SOURCE: https://artificialanalysis.ai/models/qwen3-7-max — the 56.6 vs 57 discrepancy is likely two different snapshot dates of a live-updating index)

- Cautionary precedent cited by techtimes regarding unverified Chinese-lab leaderboard claims generally: Kimi K3's **"first-place Arena.AI front-end coding ranking coexisted with a 51% hallucination rate."**
  — SOURCE: https://www.techtimes.com/articles/321158/20260721/alibabas-qwen38-max-claims-second-place-behind-fable-5-no-benchmarks-published.htm

- Dataconomy notes this fits a pattern: a prior release (Qwen3.5) made comparisons to GPT-5.2 and Claude 4.5 Opus that also "weren't independently confirmed."
  — SOURCE: https://dataconomy.com/2026/07/20/qwen3-8-24t-parameters-alibaba-ai-model-launch/

## 4. Real-world/HN reception of Qwen3.8 (secondary, but useful color)

- Hacker News thread "Qwen 3.8 Max Preview" — mixed hands-on reports: one developer says the model **"wastes too much time, goes off track, useless stuck loops, cannot debug at all,"** while another claims it performs **"above the Opus level but delivers results much faster."** A commenter also flags that separate API-endpoint tokens (distinct from subscription credits) can deplete a budget fast — **"potentially $200+ in three days."**
  — SOURCE: https://news.ycombinator.com/item?id=48966190

- Hacker News thread "Qwen 3.8" — one commenter calls it a **"benchmark princess,"** implying benchmark performance doesn't translate to real-world utility; another notes big MoE models like GLM and Kimi are **"extremely token hungry"** and feel slow in practice; general skepticism that "larger models are usually worse."
  — SOURCE: https://news.ycombinator.com/item?id=48966120

- Hacker News thread "Kimi K3, Qwen 3.8, and Anthropic's (Potential) Unravelling" — one commenter argues Chinese open-weight models were "lagging 4 months behind OpenAI/Anthropic at the beginning of the year" and are "now just 4-6 weeks behind"; another counters that benchmarks "have gotten incredibly murky" and that "nothing matches Claude Code's actual capabilities in agentic workflows" regardless of claimed parity. Distillation-from-frontier-models suspicion is raised but described as unproven.
  — SOURCE: https://news.ycombinator.com/item?id=48980019

## 5. Qwen-Image-3.0

- Launched **21 July 2026**. Official Qwen framing, from Qwen's own X/Twitter announcement (quoted verbatim from the post text as surfaced in search): **"Meet Qwen-Image-3.0 — the third generation of our foundational image generation model. If 1.0 was about 'Precision,' and 2.0 added 'Variety, Completeness, Beauty & Authenticity,' then 3.0 comes down to a single word: Real (实)."**
  — SOURCE: https://x.com/Alibaba_Qwen/status/2079906336381509659 (surfaced via search snippet; direct fetch blocked by robots.txt, so treat the quote as sourced to the search result rather than a full-page verification)

- Capabilities claimed: prompt input up to **4,500 tokens** (a "4.5×" jump from the prior generation's roughly ~1,000-token limit); native rendering across **12 languages and 20+ fonts**; text rendered legibly as small as **10 pixels**; "world knowledge" (interface/UI reproduction, live data retrieval); complex multi-panel/multi-page layouts (newspapers, storyboards, exam papers, posters, infographics).
  — SOURCE: https://finance.biggo.com/news/12c55f18-4aea-4952-9cab-f2770909cf80

- Official qwen.ai blog post exists at https://qwen.ai/blog?id=qwen-image-3.0 (confirmed as a real, indexed URL via search) but its body text could not be extracted via WebFetch (page returned only metadata/SEO tags to the fetch tool) — treat detailed capability claims above as sourced to secondary coverage (BigGo Finance, Unite.AI), not to a directly-verified primary-source quote, pending manual confirmation.
  — SOURCE: https://qwen.ai/blog?id=qwen-image-3.0 (existence only; content UNVERIFIED via this tool)

### License / weights — notable reversal from prior generations

- **No downloadable weights and no license disclosed for Qwen-Image-3.0.** This is a explicit break from precedent: **Qwen-Image 1.0 and 2.0 both shipped with open weights under Apache 2.0**; 3.0 does not.
  — SOURCE: https://www.unite.ai/alibaba-launches-qwen-image-3-0-without-benchmarks-or-weights/ ; corroborated https://finance.biggo.com/news/12c55f18-4aea-4952-9cab-f2770909cf80 ("First-generation used Apache 2.0; current release provided no licensing information")

- No benchmark scores were published alongside the release either — also a reversal, since prior versions shipped technical reports and benchmark numbers (e.g., Qwen-Image-2.0 Pro scored **57.84** on Alibaba's own "Qwen-Image-Bench," per BigGo; Qwen-Image-3.0 remains unscored on that same benchmark).
  — SOURCE: https://finance.biggo.com/news/12c55f18-4aea-4952-9cab-f2770909cf80 ; also https://www.unite.ai/alibaba-launches-qwen-image-3-0-without-benchmarks-or-weights/

- Unite.AI skepticism, quoted: **"image quality is subjective, and text rendering is precisely the axis where generators tend to look strong in hand-picked demos and weaker under systematic testing."** Without independent verification, "the only evidence a developer can act on is Alibaba's own reel of outputs."
  — SOURCE: https://www.unite.ai/alibaba-launches-qwen-image-3-0-without-benchmarks-or-weights/

### Availability

- Available for free trial via **Qwen Chat / Qwen Studio** (chat.qwen.ai) after account creation; API trials opened via **Alibaba Cloud Bailian** and the Qwen AI Platform. A standalone Qwen Studio/Qwen App trial was described as "pending" at launch.
  — SOURCE: https://finance.biggo.com/news/12c55f18-4aea-4952-9cab-f2770909cf80 ; chat.qwen.ai access corroborated by hands-on test at https://gigazine.net/gsc_news/en/20260722-qwen-image-3/

- Some coverage claims evaluations place it **"among the top in China"** and **"just below GPT Image 2"** in text-to-image assessments — flagged here as a claim from a single secondary outlet (BigGo Finance) with no methodology disclosed; treat as **weakly sourced / UNVERIFIED**, not corroborated elsewhere in this research pass.
  — SOURCE: https://finance.biggo.com/news/12c55f18-4aea-4952-9cab-f2770909cf80

### Independent hands-on testing / skepticism

- Hacker News discussion of the Qwen official announcement post: commenters report the model **"doesn't achieve qwen image 1 quality in composition or anatomical correctness,"** citing artifacts like **"third legs and glowing eyes."** Despite claims of accurate multilingual rendering, reviewers found **"Korean text is not accurate"** with spelling errors, and broken Arabic text appeared in promotional materials. Commenters also noted a characteristic "yellow tint" in outputs, speculating the model may have trained partly on prior-generation synthetic outputs rather than raw data.
  — SOURCE: https://news.ycombinator.com/item?id=48989701

- GIGAZINE (Japan) hands-on test: Japanese text rendering gave **"mixed results"** — a generated ramen-blog image contained garbled/awkward Japanese phrasing; anime/game-style illustration generation was strong ("looks just like a game screenshot") with only minor text distortion; close-up photographic texture rendering was "incredibly realistic." Overall verdict: outputs are **"practically usable"** but the model **"hasn't perfected non-English text generation"** despite the marketing claims. Small text was legible down to "as small as 10 pixels," and it supports "12 languages, including Japanese," per the same hands-on piece.
  — SOURCE: https://gigazine.net/gsc_news/en/20260722-qwen-image-3/

- A BigGo Finance headline references a report that Alibaba's Qwen official site had **"source code flooded with sexual keywords including 'Hentai'"** discovered right after the image-AI launch — surfaced in search but **not independently fetched/verified in this research pass; flag as UNVERIFIED.**
  — SOURCE (headline only, unverified): https://finance.biggo.com/news/35829630-5b15-4936-93f7-c815e98f3e68

## 6. Hugging Face downloads / derivative-model / adoption stats for the Qwen family

  — SOURCE: https://english.news.cn/20260113/004b0522f987475cbf83ffc3a8d009aa/c.html (Xinhua, Jan 2026)

Qwen **overtook Meta's Llama in cumulative downloads by October 2025**; by **December 2025** Qwen's monthly downloads exceeded the combined total of the next eight most popular models. Qwen's global download growth rate ranked first among five major open-source model vendors since H2 2025.
  — SOURCE: https://english.news.cn/20260113/004b0522f987475cbf83ffc3a8d009aa/c.html

Same source separately reports the **Qwen (chat) app had 10 million downloads in its first beta week (November 2025)** — distinct from model-repo downloads — and cites **90,000 enterprise clients** using Qwen APIs/on-prem deployments, per Alibaba's own reporting (enterprise figure not independently verified).
  — SOURCE: https://www.aicerts.ai/news/alibaba-qwen-model-downloads-metrics-and-enterprise-impact/

- A **"one billion downloads"** claim appears in a July 2026 outlet (published 10 Jul 2026, updated 13 Jul 2026), asserting Qwen "officially crossed one billion cumulative downloads," overtaking Llama — but this piece **provides no exact figure, no link to underlying Hugging Face data, and no named source beyond "platforms like Hugging Face."**
